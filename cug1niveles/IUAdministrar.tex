\subsection{Pantalla: Gestionar Niveles}

\subsubsection{Objetivo}
  Mostrar la información correspondiente a los Niveles que pertenecen a un Programa con el fin de mantener actualizada la información, ver Figura~\ref{IUGestNiveles}. Viene de Menú Gestión de Programas.

%\subsubsection{Diseño}%[h]
\IUfig[0.6]{CUD1-1/gestionarNiveles.png}{IUGestNiveles}{Gestionar Niveles}	

\subsubsection{Salidas}
  En esta pantalla se muestran los Datos de los Niveles registrados en una tabla, ordenados por Posición en forma ascendente.

\subsubsection{Comandos}
\begin{itemize}
 \item Al presionar \IUbutton{\includegraphics[scale=0.15]{images/icons/agregar.png}} el sistema muestra la pantalla \IUref{IUAgregarNivel}{Agregar Nivel} para poder agregar un Nivel.
 \item Al presionar \IUbutton{\includegraphics[scale=0.1]{images/icons/editar.png}} el sistema muestra la pantalla \IUref{IUModificarNivel}{Modificar Nivel} para poder modificar un Nivel.
 \item Al presionar \IUbutton{\includegraphics[scale=0.1]{images/icons/eliminar.png}} el sistema muestra la pantalla \IUref{IUEliminarNivel}{Eliminar Nivel} para poder eliminar un Nivel.
\end{itemize}
