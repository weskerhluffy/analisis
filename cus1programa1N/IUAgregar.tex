\subsection{Pantalla: Agregar Programa}

\subsubsection{Objetivo}
Registrar un nuevo Programa, ver Figura~\ref{IUAgregarPrograma1N}. Viene de Gestionar Programas.

\IUfig[0.6]{CUG1/AgregarPrograma1N.png}{IUAgregarPrograma1N}{Agregar Programa.}

\subsubsection{Entradas}
Datos del Programa:
\begin{itemize}
	\item Siglas
	\item Nombre
	\item Duración
	\item Fecha Inicio
	\item Fecha Fin
	\item Resumen
\end{itemize}

\subsubsection{Comandos}
\begin{itemize}
	\item \IUbutton{Aceptar}: Tras ingresar los datos esta opción registra el nuevo Programa siempre que sus datos sean correctos. En otro caso el sistema mostrará los mensajes MSG1 (por la regla \BRref{RN1}). Al terminar el sistema regresa a la pantalla \IUref{IUGestProgramas1N}{Gestionar Programas}.
	\item \IUbutton{Cancelar}: Cancela el registro del nuevo Programa y regresa a la pantalla \IUref{IUGestProgramas1N}{Gestionar Programas}.
\end{itemize}
