\subsection{Pantalla: Gestionar Acciones}\label{pantalla:EditarProyectoAccion}
La pantalla de editar cuenta con la posibilidad de gestionar las acciones asi como los presupuestos. En la pantalla Editar Proyecto \ref{pantalla:EditarProyectoPresupuesto} se muestran las caracteristicas correspondientes al presupuesto.

\subsubsection{Objetivo}
  Mostrar la información correspondiente a las Acciones que tienen registradas los Proyectos, ver Figura~\ref{IUGestAcciones}. Viene de Menú Gestión de Proyecto.

%\subsubsection{Diseño}%[h]
\IUfig[0.6]{cuc2-2-1acciones/gestionarAcciones.png}{IUGestAcciones}{Gestionar Acciones}	

\subsubsection{Salidas}
  En esta pantalla se muestran los Datos de los  registros en una tabla.

\subsubsection{Comandos}
\begin{itemize}
 \item Al presionar \IUbutton{\includegraphics[scale=0.15]{images/icons/agregar.png}} el sistema muestra la pantalla \IUref{IUAgregarAccion}{Agregar Accion} para poder agregar una Acción.
 \item Al presionar \IUbutton{\includegraphics[scale=0.1]{images/icons/Modificar.png}} el sistema muestra la pantalla \IUref{IUModificarAccion}{Modificar Accion} para poder modificar una Acción.
 \item Al presionar \IUbutton{\includegraphics[scale=0.1]{images/icons/eliminar.png}} el sistema muestra la pantalla \IUref{IUEliminarAccion}{Eliminar Accion} para poder eliminar una Acción.
\end{itemize}
