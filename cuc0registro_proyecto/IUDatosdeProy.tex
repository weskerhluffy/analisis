\subsection{Pantalla: Datos del Proyecto}

\subsubsection{Objetivo}
El objetivo de esta pantalla es mostrar los datos de un Proyecto pre-registrado y muestra al Coordinador la opción de asociar al Proyecto.

\IUfig[0.6]{cuc0/DatosdeProy.png}{IUDatosdeProy}{Pantalla Datos del Proyecto.}

\subsubsection{Salidas}
\begin{itemize}
\item Ninguna.
%  \item Datos del Proyecto Pre-registrado.
%  \item Lista de Ejes Temáticos.
%  \item Lista de Temas Transversales.
%  \item Lista de Programas.
%  \item El mensaje de operación exitosa una vez Re-registrado el proyecto.
\end{itemize}

\subsubsection{Entradas}
\begin{itemize}
 \item Datos del Proyecto Pre-registrado. 
 \item Identificador del Eje Temático seleccionado.
 \item Identificador del Tema Transversal seleccionado.
 \item Identificador del Programa seleccionado.
\end{itemize}

\subsubsection{Comandos}
\begin{itemize}
 \item \IUbutton{Asociar proyecto}: Permite editar los datos del proyecto. \IUref{IUEditarProyectos}{Gestión de Proyectos}.
 \item \IUbutton{Cancelar}: Regresa a la pantalla \IUref{IURegistrarProyecto}{Registro de Proyectos}.
\end{itemize}
