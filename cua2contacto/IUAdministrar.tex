	El mapa de navegación se muestra en la Figura~\ref{fig:mapaNavegacionCUA2}

  \begin{figure}[hbpt!]
		\centering
			\includegraphics[width=0.8\textwidth]{images/CUA2/MapaNavegacion.png}
		\caption{Mapa de navegacion para el CU A2 Gestion de Contactos.}
		\label{fig:mapaNavegacionCUA2}

\subsubsection{Objetivo}
Mostrar el menú para realizar altas, bajas o cambios de los Contactos de algun Usuario ver Figura~\ref{IUGesContactos}.


\IUfig[0.6]{CUA2/GestionContactos.png}{IUGesContactos}{Gestionar Contactos.}

\subsubsection{Salidas}
Contactos registrados de un Usuario.

\subsubsection{Entradas}
Ninguna.


\subsubsection{Comandos}
\begin{itemize}
 \item \IUbutton{Agregar Contacto} Al presionar este botón se Muestra la ventana \IUref{IUAgregarContacto}{Agregar Contacto}.
 \item \IUbutton{\includegraphics[height=10pt]{images/icons/Modify.jpg}} Permite realizar modificaciones en el registro seleccionado, muestra la pantalla \IUref{IUModificarContacto}{Modificar Contacto.}.
 \item \IUbutton{\includegraphics[height=10pt]{images/icons/eliminar.png}} Muestra la ventana \IUref{IUEliminarContacto}{Eliminar Contacto.} donde se pueden eliminar los contactos ya registrados.
 \item \IUbutton{Cancelar} Al presionar este botón regresa a la pantalla del Caso de Uso Gestionar Usuarios.
\end{itemize}
