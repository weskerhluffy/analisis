\subsection{Pantalla: Agregar Contacto}

\subsubsection{Objetivo}
Registrar un nuevo Contacto para un Usuario Viene de \IUref{IUGesContactos}{Gestionar Contactos.} ver Figura~\ref{IUAgregarContacto}.


\IUfig[0.6]{CUA2/AgregarContactos.png}{IUAgregarContacto}{Agregar Contacto}

\subsubsection{Salidas}
Mensaje de operación exitosa.

\subsubsection{Entradas}
Tipo de Contacto, Valor, prioridad,descripción.


\subsubsection{Comandos}
\begin{itemize}
 \item \IUbutton{Aceptar}: Al presionar este botón registra los datos de un nuevo Contacto y regresa a la pantalla \IUref{IUGesContactos}{Gestionar Contactos}
 \item \IUbutton{Cancelar}: Cancela la operación y regresa a la pantalla \IUref{IUGesContactos}{Gestionar Contactos} sin realizar ninguna modificación en los contactos.
\end{itemize}
