\subsection{Pantalla: Eliminar Usuario}

\subsubsection{Objetivo}
Permite al usuario eliminar un Coordinador seleccionado.

\IUfig[0.6]{CUDG2/EliminarCoordinador.png}{IUEliminarCoordinador}{Eliminar Usuario}	

\subsubsection{Salidas}

\begin{itemize}
 \item Datos del Coordinador que se desea eliminar \ref{dd:DatosEditablesCoordinador}. Estos campos no pueden ser modificados por el usuario.
 \item Se muestra el mensaje (MSG-4) \ref{MSG4}.
\end{itemize}


\subsubsection{Comandos}
\begin{itemize}
 \item \IUbutton{Aceptar} Cambia el estado del Coordinador a incativo y regresa a la pantalla \IUref{IUGestCoordinadores}{Gestionar Usuarios}.
 \item \IUbutton{Cancelar} Cancela la operaci√≥n. Lleva a la pantalla \IUref{IUGestCoordinadores}{Gestionar Usuarios} y no elimina el registro.
\end{itemize}
