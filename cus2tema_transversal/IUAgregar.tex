Datos Tema Transversal \ref{dd:TemaTransversal}.
\begin{itemize}
 \item Nombre del Tema Transversal.
 \item Descripción del Tema Transversal.
\end{itemize}

\subsection{Salidas}
Se muestra el mensaje \ref{MSG4}
\subsubsection{Comandos}
\begin{itemize}

 \item \IUbutton{Cancelar} Cancela la operación. Lleva a la pantalla \IUref{IUGestTemasTransversales}{Gestionar Temas Transversales}
