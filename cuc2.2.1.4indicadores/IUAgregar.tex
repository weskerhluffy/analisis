\subsection{Pantalla: Agregar Indicador}

\subsubsection{Objetivo}
Registrar un nuevo Indicador, ver Figura~\ref{IUAgregarIndicador}. Viene de Gestionar Indicadores.

\IUfig[0.8]{cu2-2-1-4indicador/Agregar.png}{IUAgregarIndicador}{Agregar Indicador.}

\subsubsection{Salidas}

 Lista de Unidades.

\subsubsection{Entradas}
 Datos del Indicador\ref{dd:Indicador}:
 \begin{itemize}
  \item Pregunta.
  \item Tipo.
  \item Meta.
  \item Unidad.
  \item Peso.
 \end{itemize}

\subsubsection{Comandos}
\begin{itemize}
 \item \IUbutton{Aceptar}: Registra los datos de un nuevo Indicador y lleva a la pantalla \IUref{IUEvaluarAccion}{Gestión de Indicadores Físicos}.
 \item \IUbutton{Cancelar}: Regresa a la pantalla \IUref{IUEvaluarAccion}{Gestión de Indicadores Físicos}.
\end{itemize}
