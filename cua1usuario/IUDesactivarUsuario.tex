\subsection{Pantalla: Desactivar Usuario y reasignar trabajo}
Pantalla emergente que permite al Administrador reasignar el Trabajo de un Usuario al ser desactivado.
\subsubsection{Objetivo}
Reasignar el trabajo de un Usuario que ha sido \textit{Desactivado} a un nuevo Usuario que tenga el mismo Perfil y que se encuntre \textit{Activo}. Figura~\ref{IUReasignarTrabajo}. Viene de \IUref{IUModificarUsuario}{Modificar Usuario.}.

\IUfig[0.6]{CU1/ReasignarTrabajo.png}{IUReasignarTrabajo}{Pantalla: Reasignar trabajo de un Usuario.}

\subsubsection{Salidas}
Lista de Usuarios con el mismo Perfil que el Usuario Desactivado y que se encuntren Activos.

\subsubsection{Entradas}
Identificador del usuario seleccionado.
\end{itemize}

\subsubsection{Comandos}
\begin{itemize}
 \item \IUbutton{Aceptar}: Asigna el trabajo al Usuario seleccionado de la lista de usuarios y regresa a la pantalla \IUref{IUModificarUsuario}{Modificar Usuario.}.
 \item \IUbutton{Cancelar}: Regresa a la pantalla \IUref{IUModificarUsuario}{Modificar Usuario.}.
\end{itemize}
