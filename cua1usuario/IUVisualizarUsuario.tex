\subsection{Pantalla: Consulta de Usuario}
%IUAgregarPerfil
\subsubsection{Objetivo}
El Mensaje de Consulta de Usuario. Figura~\ref{IUVisualizarUsuario}. Aparece Sobre \IUref{IUGestUsuarios}{Géstionar Usuarios.}.


\IUfig[0.6]{CU1/VisualizarUsuario.png}{IUVisualizarUsuario}{Consulta de Usuario.}

\subsubsection{Salidas}
Datos actuales pertenecientes al registro del Usuario que se pretende modificar\ref{dd:Usuario}:
\begin{itemize}
 \item Login
 \item Contraseña
 \item Nombre
 \item Apellido Paterno
 \item Apellio Materno
 \item RFC
 \item Cargo
 \item Dirección
\end{itemize}

\subsubsection{Comandos}
\begin{itemize}
 \item \IUbutton{OK}: Regresa a la pantalla \IUref{IUGestUsuarios}{Géstionar Usuarios.}.
\end{itemize}
