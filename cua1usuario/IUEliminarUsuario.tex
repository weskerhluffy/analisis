\subsection{Pantalla: Eliminar Usuario}
%IUAgregarPerfil
\subsubsection{Objetivo}
El Mensaje de Eliminar un Usuario. Figura~\ref{IUEliminarUsuario}. Aparece Sobre \IUref{IUGestUsuarios}{Géstionar Usuarios.}


\IUfig[0.6]{CU1/EliminarUsuarios.png}{IUEliminarUsuario}{Eliminar Usuario.}

\subsubsection{Salidas}
Datos actuales pertenecientes al registro del Usuario que se pretende modificar\ref{dd:Usuario}:
\begin{itemize}
 \item Login
 \item Contraseña
 \item Nombre
 \item Apellido Paterno
 \item Apellio Materno
 \item RFC
 \item Cargo
 \item Dirección
\end{itemize}

\subsubsection{Comandos}
\begin{itemize}
 \item \IUbutton{Aceptar}: Elimina los datos de un Usuario y regresa a la pantalla \IUref{IUGestUsuarios}{Géstionar Usuarios.}.
 \item \IUbutton{Cancelar}: Regresa a la pantalla \IUref{IUGestUsuarios}{Géstionar Usuarios.}.
\end{itemize}
