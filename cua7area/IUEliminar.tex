\subsection{Pantalla: Eliminar Área}

\subsubsection{Objetivo}
Eliminar un Área seleccionada. Vease Figura ~\ref{IUEliminarArea}, viene de Gestionar Áreas.

\IUfig[0.6]{CU7/EliminarArea.png}{IUEliminarArea}{Eliminar Área.}

\subsubsection{Salidas}
Datos del Área seleccionada, los cuales se encuentran deshabilidados.
\begin{itemize}
 \item Nombre del área
 \item Siglas del área
 \item Descripción del área
\end{itemize}
Muestra MSG4 \ref{MSG4}

\subsubsection{Comandos}
\begin{itemize}
 \item \IUbutton{Aceptar} Elimina del sistema el área que el usuario ha seleccionado. Cuando el área tenga datos asociados mostrará el MSG6.7 \ref{MSG6.7}. 
 \item \IUbutton{Cancelar}: Regresa a la pantalla \IUref{IUGestAreas}{Gestionar Áreas}.
\end{itemize}
